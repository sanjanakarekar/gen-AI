# AI Product Intelligence System – GenAI Workshop

This repository contains my solutions for the **GenAI Workshop** tasks focused on enhancing an AI-powered Product Intelligence System for e-commerce platforms.

## Task 1: Smart Product Recommendation Engine

Developed a recommendation system that suggests **complementary products** frequently purchased together, going beyond visual similarity. Built a simple Streamlit interface to search for a product and display relevant recommendations with product images.

## Task 2: Unique Product Catalog Creation

Created a system to identify and remove **duplicate or near-duplicate products**, generating a clean and unique product catalog. This helps improve product organization and reduces redundancy in online marketplaces.

## Task 3: Reverse Product Search

Implemented a **text-based product search** system that allows users to search for products using keywords or product names instead of images. The application matches user queries with product information and returns the most relevant products.
